Контроль : клиент

 

xС осуществляет автоматический контроль введенных данных на стороне клиента а также обязательность полей для заполнения.

Контроль проводится посредством сценария local-lib.js.

Контроль осуществляется перед отправкой формы на сервер.

Группы контролируемых полей:

required*

- контроль заполненности поля

ogrn

- форматно-логический контроль и контроль по контрольной сумме

inn

- форматно-логический контроль и контроль по контрольной сумме

tnved

- допускаются только целые числа

float

- контроль разделителя целой и десятичной части. допускаются только цифры

int

- допускаются только цифры

*- технически не является группой полей. Действует для всех групп.

В случае обнаружения ошибки в приведенных в таблице выше полях отправка формы будет приостановлена, ошибочное поле - подсвечено и выведено соответствующее сообщение.

Также в самом низу формы, рядом с кнопкой отправки будет выведено дополнительное сообщение об ошибке.
