<textblock> Блок текста вне таблицы

 

<textblock>...</textblock> - В данный контейнер помещается описание логической ячейки, находящейся внутри физической ячейки таблицы (<td>)

Блоки делятся на блоки данных и блоки разметки. Блоки данных содержат описание внешнего вида вывода одного поля базы данных из одного ряда данных базы. Блоки разметки используются для визуального разделения логических ячеек внутри одной физической ячейки таблицы (<td>).

Блок данных:

 

   <textblock type="before/after">     
      <style>стили и классы текстового блока</style>
      <template>данные для вывода</template>
   </textblock>

Составляющие блока:
 

Элемент

Обязательность

Описание

 <style>

нет

Описание стилей и классов, применяемых к родительскому элементу (<span>).

 type=""

да

Положение текстового блока: before - непосредственно до таблицы, after - сразу после.

 <template>

нет

Строка, в которой выводится текст.

Доступны MACRO, PARSER.

Недоступна системная переменная VALUE.

 

Пример:

 

 <textblock type="after">
         <style>
                 <decor>b</decor>
                 <inline></inline>
                 <class></class>
         </style>
         <template>Всего записей: %MACRO{number(total_rows)}%</template>
 </textblock>
