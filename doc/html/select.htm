select. Выпадающий список

 

Элемент интерфейса в виде раскрывающегося списка, а также список с одним или множественным выбором.

 

        <control_group>select</control_group>

Существует два вида описания данного элемента.

•Выпадающий список с предопределенными опциями.

•Выпадающий список с динамически подгружаемыми опциями.

Выпадающий список с предопределенными опциями.
 

<control>
        <control_group>select</control_group>
        <control_name>test_select</control_name>
        <control_id>radio_select</control_id>
        <control_label>Тестовый селект</control_label>
        <control_default_value>2</control_default_value>
                <control_options>
                        <option>
                                <label>sel1</label>
                                <value>1</value>
                        </option>
                        <option>
                                <label>sel2</label>
                                <value>2</value>
                        </option>
                        <option>
                                <label>sel3</label>
                                <value>3</value>
                        </option>
                </control_options>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.

Выпадающий список с динамически подгружаемыми опциями.
 

<control>
        <control_group>select</control_group>
        <control_name>firmsend_country</control_name>
        <control_label>Страна отправления товара</control_label>
        <control_sql>SELECT `name`,`alfa2` FROM `kw` ORDER BY `name` ASC</control_sql>
        <control_default_value>RU</control_default_value>
        <control_classes>select_country</control_classes>
</control>

В элементе <control_sql> указывается запрос на выборку из базы значений label и value  для опций списка. Причем, первый выбранный столбец используется для label , а второй - для value
