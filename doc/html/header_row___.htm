<header_row> Дополнительный заголовок таблицы

 

Начиная с версии 2.3 поддерживается создание дополнительной шапки таблицы.

 

<header_row>...</header_row> -  В данный тэг заключается описание дополнительного заголовка таблицы.

 

Указывается единожды.

 

Правила формирования содержимого аналогичны правилам для <data_row>

 

Начиная с версии 2.6 допускается описание более одного элемента <row> для заголовка.
