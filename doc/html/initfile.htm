Инициализирующий файл

 

Существует возможность инициализировать некоторые поля формы при помощи заранее подготовленного инициализирующего файла.

 

Для включения опции загрузки инициализирующего файла необходимо при инициализации xControls методом ^xcontrols:init[] указать опцию $.enable_initfile(true) (см. Инициализация)

 

При указании данной опции будет сгенерирована дополнительная форма с полем загрузки инициализирующего файла.

 

Шаблон инициализирующего файла генерируется статическим методом ^generate_initfile_template[] на основе описательного XML (см. Генерация шаблона инициализирующего файла).

 

Для того, чтобы пометить элемент управления описательного XML как доступный к загрузке через инициализирующий файл, необходимо для данного элемента <control> указать тэг <accept_inifile/>.

На основе помеченных данным тэгом элементов управления и составляется шаблон.

 

Структура шаблона аналогична структуре описательного XML и имеет следующий вид:

 

<?xml version="1.0" encoding="UTF-8"?>
<document>
        <form>
                <control>
                         <control_name>имя поля 1</control_name>

                         <control_default_value>значение поля 1</control_default_value>
                </control>
                <control>

                         <control_name>имя поля 2</control_name>
                         <control_default_value>значение поля 2</control_default_value>
                </control>
               
                ...
               
                <control>

                         <control_name>имя поля n</control_name>
                         <control_default_value>значение поля n</control_default_value>
                </control>
        </form>
</document>

 

Значения, которые необходимо загрузить в поле формы, указываются в тэге <control_default_value> для соответствующего имени поля формы.

 

Изменение и/или добавление имен полей не допускается (!).

 

Инициализирующий файл всегда имеет формат XML и расширение *.xml. При попытке загрузки файла с другим расширением или форматом система выдаст сообщение об ошибке.

 

При обнаружении неизвестного системе имени поля ввода в инициализирующем файле, загрузка данных будет отменена и система выдаст соответствующее сообщение об ошибке с указанием неверного поля.
