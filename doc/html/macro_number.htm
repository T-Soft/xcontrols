number. Вывод номеров строк таблиц

Для работы с датой и временем используется макрос : number( mode )

Где, в качестве параметра mode передается одна из строк.
 

 current_row

- вернет номер текущей строки с начала таблицы

 page_row

- вернет номер текущей строки с начала текущей страницы

 total_rows

- вернет количество строк в текущей таблице

Данный макрос возвращает корректные значения только в случае вызова из модуля view.

Пример

 

 %MACRO{number(total_rows)}%

Для двадцатистрочной таблицы вернет 20.

Доступность макроса:

Модуль

Доступность
