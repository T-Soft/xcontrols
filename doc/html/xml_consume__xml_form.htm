<xml_consume> Включение XML из внешнего файла

 

Начиная с версии 2.1 поддерживается возможность включать в файл описания отображения view.xml содержимое других 

(дополнительных) xml документов при условии совпадения структуры элементов.

 

Структура элемента описания поглощаемого файла:

 

 <include_xml>
                 <prefix>префикс имен элементов дополнительного XML</prefix>

                 <file>путь к дополнительному XML</file>
 </include_xml>

Тэг используется столько раз, сколько файлов необходимо поглотить.

Составляющие:
 

Элемент

Описание

 <prefix>

Префикс, который будет прибавлен к содержимому элемента <source_name> каждого описанного в дополнительном xml поля базы данных.

Например: <source_name>foo</source_name> + <prefix>bar</prefix> = <source_name>foo_bar</source_name>

 <file>

Путь к дополнительному xml относительно главного form.xml. Например: foo.xml или ../bar.xml

 

Важно (!):

 

 Поглощаемый XML должен включать только описание ячеек таблицы <cell> или ее блоков <block>. Контроль правильности не осуществляется, заголовки не игнорируются.

 Весь текст переносится как есть с заменой имен полей на иена с префиксом.

 

Содержимое поглощаемого файла будет перенесено в то место поглощающего XML , где был указан элемент <xml_consume>.
