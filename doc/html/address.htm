address. Данные об адресе

 

Комплект полей ввода адреса. Составной тип.

 <control_group>address</control_group>

Состоит из следующих текстовых полей.

•Область

•Город

•Улица

•Дом

•Квартира

 

Пример.

 
<control>
      <control_group>address</control_group>
      <control_name>address_fieldname</control_name>
      <control_id>addr_id</control_id>
      <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.
