address_region. Данные об адресе и регионе

 

Комплект полей ввода адреса, региона из БД, ИНН, ОГРН. Составной тип.

 <control_group>address_region</control_group>

Состоит из следующих текстовых полей.

•Название

•Адрес

•Регион (выборка из БД)

•ИНН

•ОГРН

 

Пример.

 
<control>
      <control_group>address_region</control_group>
      <control_name>address_region_fieldname</control_name>
      <control_id>addr_reg_id</control_id>
      <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.
