Вызов ошибок invoke

 

В случае, когда необходимо вручную вызвать ту или иную ошибку, можно воспользоваться методом @ErrorList_$module_name$[invoke] соответствующего модуля, передав в качестве параметра $invoke$ имя ошибки, описанное в данном методе для вызываемого модуля.

 

Сообщение о некритической ошибке будет выведено на экран в верхней части страницы с конструируемой формой без указания поля, в котором возникла ошибка.

 

Также можно вызывать критические ошибки. При критической ошибке ядро завершится аварийно. (См. критические ошибки.)

 

Например при вызове.

 

^ErrorList_file[TYPE_MISMATCH]

 

Будет выведена ошибка модуля file.p "Неверный тип данных загруженного файла".

 

При вызове

 

   ^ErrorList_main[SYSTEM_CRASH]

 

Ядро завершится аварийно, так как была вызвана критическая ошибка.
