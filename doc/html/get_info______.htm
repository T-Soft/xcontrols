print_info. Вывод на экран информации о поле

 

Существует возможность получить информацию о полях базы данных. Для каждого поля предоставляется своя информация.

 

 ^print_info[uuid;fieldname;separator]

 

Где:

 

uuid

- идентификатор записи в данной таблице

fieldname

- имя поля, информацию о котором необходимо получить

separator

- разделитель между пояснительной строкой и значением свойства поля

 

Метод выводит на экран содержимое хеша, содержащего поля со специфической для данного типа поля информацией. Локализованные подписи для каждого из информационных полей хеша разделяются со значениями этих полей символом или строкой указанной в параметре separator.

 

Данный метод реализован для следующих групп элементов управления:

 

•file

 

Например, при вызове данного метода для файла с заданным разделителем ":", вывод будет иметь вид:

 

Загруженный файл:<имя файла>

Размер:<размер файла>

Дата загрузки:<дата загрузки файла>
