get_info. Получение информации о поле

 

Существует возможность получить информацию о полях базы данных. Для каждого поля предоставляется своя информация.

 

 ^get_info[uuid;fieldname]

 

Где:

 

uuid

- идентификатор записи в данной таблице

fieldname

- имя поля, информацию о котором необходимо получить

 

Метод возвращает хеш, содержащий поля со специфической для данного типа поля информацией а также служебное поле $._label , содержащее локализованные подписи для каждого из предыдущих информационных полей.

 

Данный метод реализован для следующих групп элементов управления:

 

•file

 

Например, для файла возвращаемый хеш имеет следующий вид:

 

$hRet[
      $.filename[<имя файла>]
      $.size[<размер файла>]
      $.dt_upload[<дата загрузки файла>]
      $._label[
         $.filename[Загруженный файл]
         $.size[Размер]
         $.dt_upload[Дата загрузки]
      ]
]
