<block> Описание блока ячейки таблицы

 

<block>...</block> - В данный контейнер помещается описание логической ячейки, находящейся внутри физической ячейки таблицы (<td>)

Блоки делятся на блоки данных и блоки разметки. Блоки данных содержат описание внешнего вида вывода одного поля базы данных из одного ряда данных базы. Блоки разметки используются для визуального разделения логических ячеек внутри одной физической ячейки таблицы (<td>).

Блок данных:

 

 <block type="data">     
         <style>стили и классы логической ячейки</style>
         <source_name>имя поля БД</source_name>
         <control_group>группа элемента управления из доступных xcontrols</control_group>
         <label>заголовок таблицы для данной ячейки</label>
         <format>формат вывода данных</format>         
         <template>шаблон вывода данных</template>
 </block>

Составляющие блока:
 

Элемент

Обязательность

Описание

 <style>

нет

Описание стилей и классов, применяемых к родительскому элементу (<span>).

 <source_name>

да

Имя столбца таблицы базы данных, данные из которого нужно отобразить в данной логической ячейке.

 <control_group>

да

Тип (группа) элемента управления (int, inn, ogrn и т.д.), доступного классу xcontrols.

 <label>

нет

Строка, помещаемая в заголовок таблицы для соответствующего данной логической ячейке столбца.

 <format>

нет

Формат преобразования "сырых" данных - непосредственного содержимого соответствующей <source_name> ячейки БД. Может влиять на значащую часть данных, отображаемых в логической ячейке.

 <template>

нет

Шаблон, накладываемый на данные после их форматирования.

 

Элемент <format>

Для каждой группы элементов управления, доступных xcontrols определен минимум один формат преобразования исходных данных из ячейки таблицы к отображаемому виду. (См. Форматы отображения данных).

Может быть указано только одно преобразование.

Элемент <template>

Для каждого значения, выводимого в таблицу возможно указание шаблона, который будет наложен на отоформатированные данные или информации, которая будет выведена в случае, если данные отсутствуют (См. Шаблон отображения данных).

Блок разметки:

 

 <block type="layout">
         <layout_type>hr</layout_type>
         <layout_template></layout_template>
 </block>

Составляющие блока:
 

Элемент

Обязательность

Описание

 <layout_type>

да

Тип элемента разметки

 <layout_template>

нет

Формат вывода элемента разметки.

Элемент <layout_type>

Определяет тип элемента разметки.

На текущий момент существует всего два типа: hr и br

Блок сокращенной разметки:

<block type="hr"/>

<block type="br"/>

С версии [2.6] доступны сокращенные блоки разметки. На место этих блоков xControls поместит соответствующие html тэги.

Блок служебных данных:

 

 <block type="nodata">

         <style>стили и классы логической ячейки</style>
         <label>заголовок данного элемента</label>
         <template>макросы и шаблоны</template>
 </block>
