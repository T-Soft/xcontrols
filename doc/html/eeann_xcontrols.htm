Класс xControls

Модульная система xControls используется для автоматической генерации страниц и форм на основе одного описательного XML файла и статического класса - обработчика (модуль form).

Также с версии 2.0 система поддерживает табличное отображение полей базы данных на основе описательного XML (модуль view). Наличие описательного XML для формы при этом необязательно.

Примечание: Модуль отображения поддерживает базы любой структуры. Однако, для корректной работы данной функции настоятельно рекомендуется, чтобы структура таблицы и системы хранения файлов была идентична создаваемой xcontrols.

Для управления системой используется статический класс xcontrols.
