Пользовательские сценарии *.js

 

Для использования пользовательского кода на языке JavaScript необходимо этот код разместить в директории /xc/js/xcontrols_modules/

 

Весь код должен быть обернут в следующую функцию

 

   function <название модуля>_init(){
      код функции
   }

 

Загрузка сценария происходит после загрузки соответствующего *.p модуля из директории /xc/p/xcontrols_modules/

 

Название *.p - модуля и *.js - модуля должны совпадать.
