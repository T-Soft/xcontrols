update_entry. Изменение данных на сервере

 

Обновляет данные текущей формы в базе данных по uuid записи.

 

 ^xcontrols:update_entry[hHidden_fields;hParam]

 

Где:

 

hHidden_fields

- хеш, для передачи скрытых полей, отсутствующих на форме.

hParam[

- хеш, параметров вызова.

       $.is_archive(true)

]

- при передаче параметром true, сохраняет предыдущее состояние полей записи БД в архивную таблицу.

 

Тип возвращаемого значения:        hash.

 

Параметр hHidden_fields:

 

 Скрытые поля передаются в виде $.key[value] , где key - имя поля БД, а value - его содержимое.

 

 В случае совпадения имени переданного таким образом поля ($.key) и поля, получаемого из формы, последнее будет перезаписано переданным в дополнительном хеше значением value.

 

При подготовке к сохранению данных в базе, для каждого поля формы будет вызван метод ^xcontrols:db_preprocess_$sControl_group[].

 

 $sControl_group                - группа, к которой принадлежит элемент управления.

 

Данный метод, может реализовывать форматно-логический контроль значения поля. (См. Контроль полей формы : Сервер)
