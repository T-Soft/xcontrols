sql. Получение строки из базы данных

Для получения строки из базы данных по выполнению запроса используется макрос : sql( query )

Где, в качестве параметра query передается SQL-запрос, который должен вернуть одну строку.

Пример
 

 %MACRO{sql(SELECT `file_test` FROM `skk_results` WHERE `uuid`='466EF12D-DBC2-4E52-9E19-4DAE202D80F4')}

Вернет имя фйала для данного uuid.
