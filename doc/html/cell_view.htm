<cell> Описание ячейки таблицы

 

<cell rowspan="n" colspan="m">...</cell> - В данный контейнер помещается описание одной ячейки таблицы (<td>)

Указывается столько раз, сколько ячеек (<td>) отводится на один ряд таблицы (<row> - <tr>).

Составляющие:

 

 rowspan="n"

n - число, показывающее, сколько ячеек нужно объединить в данную [2.3]

 colspan="m"

m - число, показывающее, сколько строк объединить в данную, ячейку. [2.3]

 <style>

- описание стилей и классов, применяемых к родительскому элементу (<td>). Необязательный элемент.

 <block>

- описание логической ячейки, находящейся внутри физической ячейки таблицы (<td>)

 <header>

- название ячейки, которое будет выводиться принудительно вместо конкатенации названий блоков

 <noheader/>

- тэг, указывающий, что заголовочную ячейку для данного столбца создавать не нужно. Используется вместе с rowspan. [2.3]
