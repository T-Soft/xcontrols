Критические ошибки ядра

 

При критической ошибке ядро завершается аварийно с указанием типа критической ошибки.

 

Список критических ошибок ядра:

 

Ошибка

Имя

Описание

Сообщение

xcontrols.crash

SYSTEM_CRASH

Неизвестная критическая ошибка

xControls core module -> System crashed

xcontrols.crash_edit

CRASH_EDIT_ENTRY

Попытка загрузить из базы несуществующую запись

xControls core module -> editting non existing entry

xcontrols.crash_update

CRASH_UPDATE_ENTRY

Попытка обновить в базе несуществующую запись

xControls core module -> updating non existing entry

xcontrols.crash_invoke

CRASH_INVALID_INVOKE

Попытка вручную вызвать несуществующий класс ошибки

xControls core module -> invoking non existing error

xcontrols.crash_deserialize

CRASH_NO_FAILED_ID

Попытка десериализовать несуществующий файл*

xControls core module -> deserializing non existing file

xcontrols.crash_invalid_field

CRASH_UNKNOWN_FIELD_INFO

Попытка получить информацию о неизвестном поле

xControls core module -> getting unknown field info

 

Все критические ошибки содержат в своем имени слово CRASH.

 

* - функция десериализации после успешного завершения удаляет временный файл. Попытка обновить страницу приведет к ошибке CRASH_NO_FAILED_ID.
