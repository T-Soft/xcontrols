form. Получение значения поля формы

Для получения значения поля формы используется макрос : form( fieldname )

Где, в качестве параметра fieldname передается строка - имя поля, к которому нужно осуществить доступ.

Пример

 

 при http://xcontrols.dev/add/?uuid=123456

 

 %MACRO{form(uuid)}%

Вернет 123456
