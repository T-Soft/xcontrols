file_ajax. Загрузка файла по частям [недоступно]

 

В ПРОЦЕССЕ РАЗРАБОТКИ

 

Поле для присоединения и отправки файла по частям.

 

Корректность введенного значения (совпадение с типом или расширением) проверяется на сервере. В случае ошибки форма возвращается пользователю для исправления.

Контроль данного поля аналогичен контролю поля file.

 

         <control_group>file_ajax</control_group>

Пример.
 

<control>
        <control_group>file_ajax</control_group>
        <control_name>tst_file</control_name>
        <control_id>file_id</control_id>
        <file_type>file_type</file_type>
        <file_ext>file_extensions</file_ext>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.
 
Визуально данное поле выглядит как кнопка с надписью "Загрузить файл". После нажатия на данную кнопку открывается модальное окно загрузки частей файла.

Для передачи частей файла используется AJAX. Окно невозможно закрыть, пока передача частей не будет завершена успешно или отменена.

В случае успешной передачи всех частей файла на сервер осуществляется сборка файла из частей и сохранение его по тем же правилам, что и файл из обычного поля file.
