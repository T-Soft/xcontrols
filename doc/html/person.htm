person. Личные данные

 

Комплект полей ввода личных данных. Составной тип.

 

   <control_group>person</control_group>

Состоит из следующих текстовых полей.

•ФИО

•Вид документа, удостоверяющего личность

•Серия и номер документа

•Кем выдан документ

•Дата выдачи

•Номер телефона

•Почтовый адрес

•E-mail

Пример.
 

<control>
        <control_group>person</control_group>
        <control_name>person_fieldname</control_name>
        <control_id>prsn_id</control_id>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.
