text. Простой текст

 

Незначащее поле.
 

        <control_group>text</control_group>

Пример.
 

<control>
        <control_group>text</control_group>

         <control_type>Тип выравнивания выводимого текста</control_type>
        <control_label>Текст, который необходимо вывести</control_label>
        <control_id>addr_id</control_id>
        <control_classes>foo</control_classes>
</control>

Элементы <control_type>,<control_id> и <control_classes> являются необязательными.

Поскольку основой создаваемого интерфейса является двухстолбчатая таблица, можно выбирать в какой из колонок этой таблицы выводить текст.

Для этого необходимо в тэге <control_type> указать одно из следующих значений:

left

right

both

Если тэг оставить пустым по умолчанию будет принято значение both.

Если указать тип выравнивания, не предусмотренный модулем, будет вызвана ошибка .INVALID_TEXT_ALIGN_TYPE - Неверное значение в поле выравнивания текста.
