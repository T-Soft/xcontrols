Ошибки ядра

 

К ошибкам ядра относятся все некритические ошибки модулей, не обрабатываемые соответствующим методом, а также некритические ошибки самого ядра, описанные в его методе @ErrorList_$module_name$[invoke].

 

Список некритических ошибок ядра:

 

Имя

Описание

Сообщение

INVALID_VALUE

Обнаружено неверное значение

Неверный тип или значение в поле ввода или загрузки файла

UNKNOWN_ERROR

Неизвестная некритическая ошибка ядра

Неизвестная ошибка ядра xControls

UNKNOWN_FIELD

Попытка обратиться к несуществующему полю формы

Неизвестное системе поле ввода

UNKNOWN_INITFILE_FIELD

В инициализирующем фале обнаружено неизвестное поле

Неизвестное системе поле ввода в инициализирующем файле

INVALID_INTFILE_FORMAT

Попытка загрузить неверный инициализирующий файл

Недопустимый формат инициализирующего файла

CONSUME_FILE_NOT_FOUND

Попытка поглотить несуществующий документ элементом <xml_consume>

Файл для поглощения не найден по заданному пути
