Модуль VIEW

 

Модуль отвечает за просмотр записей из базы данных в виде таблицы с опициональным скроллером.

 

 

С версии 2.5 при формировании таблицы во вставках на языке Parser доступна переменная:

 

    $xcontrols:tData - текущая таблица по которой формируется view с текущем рядом который формируется во view сейчас. Состоит из полей описанных в XML в разделах <table> и <additional> .

 

Пример использования:

 

 <template>
                   <![CDATA[
                 Меры к <b>продавцу</b> завершены %PARSER{$xcontrols:tData.fixpay_date_fix}%
         ]]>
 </template>
