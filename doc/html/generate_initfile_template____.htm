generate_initfile_template. Генерация шаблона инициализирующего файла

 

Для генерации шаблона инициализирующего файла (см. Инициализирующий файл) на основе полей описательного XML , помеченных тэгом <accept_inifile/> используется метод :

 

 ^generate_initfile_template[form_id;save_path;filename]

 

Где:

 

form_id

- имя формы, для которой происходит генерация шаблона.

save_path

- путь, по которому следует сохранить шаблон. По умолчанию равен $.temp_path из метода ^init[]

filename

- имя шаблона. По умолчанию равно "initfile_template"

 

Шаблон имеет кодировку UTF-8. Структура шаблона аналогична структуре  описательного XML.
