Замена HTML сущностей

 

Препроцессор позволяет помещать в XML для последующего преобразования в HTML сущности, такие как непечатные символы, пробелы, переносы и т.д.

 

Для указания препроцессору на необходимость вставки такой сущности используется следующий синтаксис.

 

 %ENTITY%

 

Где ENTITY - это имя HTML сущности.

 

Препроцессор заменит данную последовательность на последовательность вида

 

 &entity;

 

Со всеми экранировками, не нарушающими валидацию XML, но позволяющими корректную трансляцию в HTML.

 

Пример.

 

 Тестовая фр%NBSP%аза

 

 После преобразования в HTML будет выглядеть как

 

 Тестовая фр&nbsp;аза .
