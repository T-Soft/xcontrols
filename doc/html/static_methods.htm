Статические методы

 

В классе xcontrols реализованы следующие статические методы.

 

•header[hHeader_args] - Создание заголовков HTML страницы с путями к *.js и *.css файлам.

•form_init[hArgs] - инициализация статического класса.

•create_controls[sForm_id;sPrefix_name;sCustom_class;hInit] - создание элементов управления без создания оборачивающего тэга <form>. Т.е. создание отдельных элементов упарвления.

•create_form[sForm_id;hHidden_fields;hInit] - создание элементов управления в оборачивающем тэге <form>. Т.е. создание формы целиком.

•edit_form[sForm_id;sEntry_uuid] - редактирование введенных пользователем ранее данных через раннее сконструированную xControls форму ввода.

•store_data[hHidden_fields]- сохранение данных текущей формы в базу данных на основе описательного XML.

•update_entry[hHidden_fields;hParam]- обновление даннх текущей формы в базе данных по uuid записи.

•get_info[uuid;fieldname]- получение параметров поля базы данных.

•print_info[uuid;fieldname;separator]- вывод на экран параметров поля базы данных.

•file_info[rawData]- получение информации о файле на основе строки из базы данных.

•generate_initfile_template[sForm_id;save_path;filename]- генерация шаблона инициализирующего файла.

•view_init[hArgs] - инициализация модуля отображения данных.

•view[sProject_nick;sWhere_statement;sSort_statement]- Отображение таблицы с данными из БД.
