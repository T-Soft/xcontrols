<data_row> Описание ряда данных БД

 

<data_row>...</data_row> -  В данный тэг заключается описание ряда данных - одной строки таблицы базы данных, 

из которой производится выборка данных для отображения.

 

Указывается единожды.

Составляющие:

 

 <row>

- элемент содержит описание ряда результирующей таблицы (<tr>).
