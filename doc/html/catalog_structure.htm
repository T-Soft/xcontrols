Структура каталогов

 

Модуль состоит из файлов *.p *.xml *.js *.css, находящихся в следующих каталогах.

 

xc/  
|__js/

|  |___xcontrols_modules/

|  |  |______signed_xml.js  

|  |  |______file_ajax.js

|  |___detect.min.js

|  |___xc.js

|__p/

|  |___xcontrols_modules/ 

|  |  |______модули системы с расширением *.P

|  |___debug.p 

|  |___win1251utf8.p        

|  |___xcontrols_database.p 

|  |___xcontrols_errors.p 

|  |___xcontrols_main.p 

|  |___xcontrols_serialization.p 

|__style/

|  |___style.css 

|  |___index.cfg 

|__xc.p

xc_js/

|__jquery-x.xx.x/

|__jquery-ui-x.xx.x/

xc_style/

|___ui-1/ 

 

Категорически не рекомендуется менять приведенную структуру.
