date. Выбор даты

 

Поле выбора даты из календаря.

 

При отправке проверяется корректность введенного значения. В случае ошибки отправка формы отменяется.

 

 <control_group>date</control_group>

Пример.
 

<control>
        <control_group>date</control_group>
        <control_name>tst_date</control_name>
        <control_id>tst_id</control_id>
        <control_label>Тестовое поле выбора даты</control_label>
        <control_default_value>Тестовое значение по умолчанию</control_default_value>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id>, <control_default_value> и <control_classes> являются необязательными.

Помимо самого календаря, рядом будет выведена кнопка "Сегодня", а для необязательных полей данной группы еще и кнопка "Очистить".
