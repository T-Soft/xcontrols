Контроль полей формы

 

Модуль xC осуществляет контроль введенных и переданных данных на двух этапах :

 

•при заполнении (до отправки формы).

•перед помещением / обновлением в базе данных.

Контроль перед отправкой формы

На основе типа элемента управления xC осуществляет автоматический контроль введенных данных (средствами javascript - сценария на стороне клиента) и в случае некорректности ввода сообщает об этом пользователю. (см. Контроль : клиент)
 
Финальная стадия контроля - контроль на стороне сервера.
 
Контроль на стороне сервера - необязательная операция, реализуемая каждым модулем системы для своего типа данных. (см. Контроль : сервер)
