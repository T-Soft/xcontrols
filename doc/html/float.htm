float. Дробное значение

 

Поле ввода дробного значения с плавающей точкой.

 

При отправке проверяется корректность введенного значения. В случае ошибки отправка формы отменяется.
 

        <control_group>float</control_group>

Поддерживается ограничение диапазона вводимых значений. (см. Ограничение вводимого значения)

Пример.
 

<control>
        <control_group>float </control_group>
        <control_name> float_value </control_name>
        <control_id>tst_float </control_id>
        <control_label>Значение с плавающей точкой </control_label>    
        <bounds min="0.1" max="100.5" type="error" message="Значения от 0.1 до 100.5 недопустимы!"/>
        <control_default_value>200.8</control_default_value>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id>, <control_default_value> и <control_classes> являются необязательными.
