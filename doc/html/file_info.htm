file_info. Получение информации о файле

 

Существует возможность получить информацию файле, путь к которому хранится в базе данных.

 

 ^file_info[rawData]

 

Где:

 

rawData

- данные из поля таблицы, где хранится файл

 

Метод возвращает хеш, содержащий поля с информацией о файле.

 

Данный метод специфичен для следующих групп:

 

•file

 

Возвращаемый хеш имеет следующий вид:

 

$hRet[
      $.filename[<имя файла>]
      $.size[<размер файла>]
      $.dt_upload[<дата загрузки файла>]

      $.file_path[<путь до файла от корня>]

      $.content_type[<тип содержимого фала>]
]
