<table> Описание таблицы отображения

 

<table>...</table> - В данный элемент заключается описание таблицы, которая будет получена в результате обработки описательного XML 

отображения view.xml.

Внутри данного элемента находится элемент описания стилей и классов, применяемых к таблице целиком.

Составляющие:

 

 <style>

- элемент описания стилей и классов таблицы.

 <additional>

- дополнительные поля в выборке. В таблице не отображаются.
