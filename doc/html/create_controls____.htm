create_controls. Созадние отдельных элементов

 

Создание самостоятельных элементов управления на основе описательного XML.

 

^xcontrols:create_controls[sProject_nick;;;hInit]

 

sProject_nick                 - имя проекта (формы). (!) Должно совпадать с именем каталога, содержащего описательный XML.

hInit                         - хеш, инициализирующий поля формы. (!) Имена ключей хеша должны совпадать с именами полей формы.

 

Тип возвращаемого значения:        void.

 

(!) Использование нежелательно.
