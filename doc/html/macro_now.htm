now. Дата и время

Для работы с датой и временем используется макрос : now( mode [,offset] )

Где, в качестве параметра mode передается одна из строк.
 

time

- установка текущего времени

date

- установка текущей даты

datetime

- установка даты и времени

Необязательный параметр offset определяет смещение даты-времени от текущей вперед (значение положительное)
или назад (значение отрицательное) на указанное количество дней.

Для смещения на один час в качестве параметра указывается дробь 1/24

Макрос возвращает дату и время в формате sql-string.

Пример

 

 %MACRO{now(datetime,-1/24)}%

Вернет текущую дату-время со смещением на час назад от ткущей

 %MACRO{now(date,1)}%

Вернет текущую дату со смещением на день вперед от ткущей
