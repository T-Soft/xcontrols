header. Создание заголовков <link>, <script>

 

Для корректного отображения графического интерфейса, модулю необходим набор заголовков.

 

В заголовках указываются пути до *.js и *.css файлов.

 

 ^xcontrols:header[
         $.BP[/egov-skk/]
         $.theme[sunny]

              $.scroller(1)

         $.noprint(1)

         $.foto(1)
 ]

 

Где:

 

$.BP

- базовый путь к каталогу, содержащему директории /js и /style. По умолчанию: каталог в котором расположен главный файл библиотеки xcontrols_main.p

$.theme

- название темы оформления графического интерфейса. По умолчанию: smoothness.

$.scroller

- включение скроллера страниц - 1. По умолчанию: 0

$.noprint

- включение javascript сценария, который развешивает на все элементы <body> стиль noprint для возможности вывода страницы на печать заданным образом. По умолчанию: 0

$.foto

- подключение библиотек, скриптов и стилей для отображения фотографий. По умолчанию: 0.
