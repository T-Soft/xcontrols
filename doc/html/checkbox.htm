checkbox. Элементы выбора

 

Поле ввода в виде галочки (флажка)

 

        <control_group>checkbox</control_group>

Существует два вида описания данного элемента.

•Выпадающий список с предопределенными опциями.

•Выпадающий список с динамически подгружаемыми опциями.

На сервер значение этого поля поступает в виде 1, если флажок стоит или 0, если флажок не выставлен.

Пример.
 

<control>
        <control_group>checkbox</control_group>
        <control_name>test_checkbox</control_name>
        <control_id>tst_checkbox</control_id>
        <control_label>Тестовы чекбокс</control_label>
        <control_default_value>on</control_default_value>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.

В качестве значения по умолчанию выставляется только "on" (соответствует поставленному флажку) или не выставляется ничего.
