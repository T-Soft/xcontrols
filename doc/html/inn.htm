inn. ИНН

 

Поле ввода числового значения ИНН.

 

При отправке проверяется корректность введенного значения. В случае ошибки отправка формы отменяется.

 <control_group>inn</control_group>

Пример.
 

<control>
        <control_group>inn</control_group>
        <control_name>inn_person</control_name>
        <control_id>radio_select</control_id>
        <control_label>ИНН</control_label>  

         <control_default_value>540726661105</control_default_value> 
        <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.
