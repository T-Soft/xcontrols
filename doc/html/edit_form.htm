edit_form. Редактирование содержимого формы

 

Построение формы с заполненными данными из базы полями. Используется для редактирования ранее введенных данных.

 

 ^xcontrols:edit_form[sProject_nick;uuid]

 

Где:

 

sProject_nick

- имя проекта (формы). (!) Должно совпадать с именем каталога, содержащего описательный XML.

uuid

- идентификатор, по которому будет происходить выборка записи для редактирования.

 

Тип возвращаемого значения:        void.
