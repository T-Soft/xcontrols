view_init. Инициализация модуля отображения

 

При инициализации модуля необходимо передать следующие опции.

 

 ^xcontrols:view_init[

         $.user_view_folder_path[/.xcontrols_files/view/]
         $.rpp[records_per_page]
         $.binary_path[/folder_with_saved_files/]
 ]

 

Где:

 

$.user_view_folder_path

- путь к директории с описательным XML.

Важно: каждый описательный XML должен находиться в своем каталоге.

$.rpp

- количество записей базы данных, отображаемых на одной странице. Данные для встроенного скроллера.

$.binary_path

- путь сохранения файлов переданных через элемент <file>.

 

Важно: Все пути передаются с закрывающим " / "

Важно: Все ключи хеша имеют именно такие имена.

 

$.rpp[]

 

 Если в запросе указан параметр xcrpp="n", то n перезапишет текущее значение rpp

 

Тип возвращаемого значения:        void.
