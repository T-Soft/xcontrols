<additional> Дополнительные поля в выборке

 

Начиная с версии 2.2 поддерживается возможность включать в  набор полей, генерируемый при анализе view.xml дополнительные поля, 

не участвующие в показе, нужные для служебных целей.

 

Структура элемента описания дополнительных полей:

 

 <additional>
                 <field>

                         <source_name>имя дополнительного поля 1</source_name>                        

                 </field>

                 ...

                 <field>

                         <source_name>имя дополнительного поля n</source_name>                                

                 </field>
 </additional>

Тэг <field> используется столько раз, сколько полей необходимо добавить.

Элемент

Обязательность

Описание

 <source_name>

да

Имя столбца таблицы базы данных, данные из которого нужно включить в выборку.
