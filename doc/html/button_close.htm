button_close. Кнопка отмены редактирования

 

 <control_group>button_close</control_group>
 <control_default_value>надпись на кнопке</control_default_value>

Кнопка осуществляет перенос пользователя на страницу с url, указанным в поле $.form_target метода ^init[] без сохранения изменений в редактируемой форме.

В случае отсутствия значения у элемента <control_default_value> надпись на кнопке будет принята по умолчанию : Отмена.

Создание кнопки отмены редактирования.
 

<control>
        <control_group>button_close</control_group>
        <control_name>test_button_close</control_name>
        <control_id>tst_btn_cls</control_id> 
        <control_default_value>Надпись на кнопке</control_default_value>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id>, <control_default_value>, <control_name> и <control_classes> являются необязательными.
