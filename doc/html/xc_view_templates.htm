<template> Шаблон отображения данных

Для каждого значения, выводимого в таблицу возможно указание шаблона, который будет наложен на
отоформатированные данные или информации, которая будет выведена в случае, если данные отсутствуют.

Отформатированные данные будут вставлены в шаблон на место следующей директивы:

  

 Шаблон %VALUE[значение по умолчанию]% продолжение шаблона

 

Где:

 

Шаблон

- любая строка.

%VALUE[значение по умолчанию]%

- ссылка на отформатированные данные. Можно передавать в %PARSER{}% и %MACRO{}%

значение по умолчанию

- значение указываемое в шаблоне в случае отсутствия данных в базе.

 

Элемент %VALUE[]%:

 Пусть используется стандартное форматирование данных элемента и шаблон Значение: %VALUE[]%. В зависимости от наличия данных (val) и значения по умолчанию (default) возможны случаи:

Данные (val)

Значение по умолчанию (default)

Вывод на экран

Есть

Нет (квадратные скобки пусты [])

Значение: val

Нет

Есть

Значение: default

Есть

Есть

Значение: val

Нет

Нет (квадратные скобки пусты [])

 

 

Указание шаблона отображения:

 

 <block type="data">
         <source_name>file_test</source_name>
         <control_group>file</control_group>
         <label>Загруженный файл 1</label>
         <format>name_size</format>
         <template>Файл : %VALUE[нет файла]%</template>
 </block>

В случае отсутствия данных выведет в соответствующее поле таблицы нет файла.

Начиная с версии 2.2 данный блок поддерживает использование макросов (См. Макросы)
