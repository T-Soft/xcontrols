button. Кнопка

 

 <control_group>button</control_group>
 <control_type>тип кнопки</control_type>

Существует два типа кнопок

•Простая кнопка - тип кнопки не указывается.

•Кнопка отправки формы - тип кнопки "submit".

Создание простой кнопки.
 

<control>
        <control_group>button</control_group>
        <control_name>test_button</control_name>
        <control_id>tst_btn</control_id> 
        <control_default_value>Надпись на кнопке</control_default_value>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.

Создание кнопки отправки формы.
 

<control>
        <control_group>button</control_group> 
        <control_type>submit</control_type> 
        <control_name>test_submit</control_name>
        <control_id>tst_sbmt</control_id> 
        <control_default_value>Надпись на кнопке отправки формы</control_default_value>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id> и <control_classes> являются необязательными.
