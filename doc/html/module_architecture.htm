Модульная архитектура

 

Каждый элемент управления обрабатывается соответствующим ему модулем на языке Parser3. Также возможно подключение соответствующего элементу управления файла JavaScript со специфичным для поля кодом.

Путь до директории с обработчиками элементов управления передается классу xcontrols при инициализации, и модули обработки, находящиеся вне данной директории, использованы не будут и попытка их вызова приведет к исключению.

Важно!

Имена модулей-обработчиков должны совпадать с содержимым тэга <control_group> соответствующего раздела описательного XML.
