Замена HTML форматирования

 

Препроцессор позволяет помещать в XML для последующего преобразования в HTML тэги форматирования, такие как выделение курсивом, жирным и т.д.

 

Для указания препроцессору на необходимость вставки такого тэга используется следующий синтаксис.

 

 ~TAG~

 

Где TAG - это имя HTML тэга.

 

Важно: для элементов, подразумевающих наличие закрывающего тэга, необходимо указать ~/TAG~

 

Препроцессор заменит данную последовательность на HTML тэг

 

 <TAG>

 

Со всеми экранировками, не нарушающими валидацию XML, но позволяющими корректную трансляцию в HTML.

 

Пример.

 

 Тестовая ~B~фраза~/B~

 

 После преобразования в HTML будет выглядеть как

 

 Тестовая <B>фраза</B>
