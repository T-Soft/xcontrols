Пользовательское форматирование данных

 

В случае необходимости собственного преобразования "сырых" данных из БД перед выводом их на экран возможно подключение соответствующего модуля.

 

Для того, чтобы модуль (форматтер) был определен системой xcontrols необходимо:

 

 1) Расположение форматтера в директории /format/ рядом с директорией /view/ из $.user_view_folder_path

 2) Совпадение имени файла форматтера и формата который он реализует. Т.е. содержимое элемента <format> и имя форматтера должны совпадать.

 3) Следующая структура форматтера:

 
           @CLASS
      xcontrols
 
                   @OPTIONS
      static
      partial
 
                   @format_%имя форматирования%[hRawControlData;fieldGroup;tDataRow;format]

 

         Где:

 

%имя форматирования%

Содержимое элемента <format> для вызова данного форматирования.

hRawControlData

Хеш - "сырые" данные из базы данных.

fieldGroup

Строка - Имя группы системы xcontrols к которой принадлежит данное поле. Например : input

tDataRow

Таблица - ряд данных БД, из соответствующей колонки которого берется информация для отображения.

format

Строка - имя форматирования.

 

Параметр hRawControlData :

 

 Данный хеш имеет следующую структуру:

         

         hRawControlData[
                      $._value[value]

                                 $.fieldname_1[value_1]

                                 $.fieldname_2[value_1]

                                         ...

                                 $.fieldname_n[value_n]
                       ]

                 

                 В случае, если элемент управления представлен одним полем базы данных, в хеше будет содержаться только один ключ $._value.

                 

                 В случае представления элемента несколькими полями, их значения будут выведены в виде $.имя_поля_1[значение1] ... и т.д.

 

Пример :

 

 

@CLASS
xcontrols
 
@OPTIONS
static
partial
 
@format_uuid_url[hRawControlData;fieldGroup;tDataRow;format][sRet]
   
   $sRet[<a href="/edit/?uuid=$hRawControlData._value">$hRawControlData._value</a>]
   
   $result[$sRet]
