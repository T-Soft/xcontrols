input. Однострочное поле ввода

      <control_group>input</control_group>

Пример.

 

<control>
        <control_group>input</control_group>
        <control_name>tst_inp </control_name>
        <control_id> tst_id</control_id>
        <control_label> Тестовое простое поле ввода</control_label>
        <control_default_value>Тестовое значение по умолчанию</control_default_value>
        <control_classes>foo</control_classes>
</control>

Элементы <control_id>, <control_default_value> и <control_classes> являются необязательными.
